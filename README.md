# HospitalCare Management System

## Project Overview

This is a secure Hospital Management System built with PHP and MySQL. It allows Doctors, Patients, and Pharmacists to manage medical records through a containerized environment.

## Technologies

- **Backend:** PHP 8.2
- **Database:** MySQL 8.0
- **Environment:** Docker & Docker Compose

## Quick Setup

To run this project locally, follow these steps:

1. Clone the repository.
2. Run the command:
   ```bash
   docker-compose up -d --build
   ```
